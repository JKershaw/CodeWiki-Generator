## MCP Server for AI Integration

Enable AI assistants like Claude Code to query your wiki for development context.

### Starting the Server

```bash
npm run mcp-server
# Or with custom wiki path
node mcp-server.js --wiki ./wikis/your-project
```

### Available Tools

1. **query_wiki** â€“ Search documentation with intelligent context gathering
2. **request_documentation** â€“ Queue documentation for missing topics

### Claude Code Integration

Configure in your Claude Code settings:

```json
{
  "mcpServers": {
    "codewiki": {
      "command": "node",
      "args": ["/path/to/CodeWiki-Generator/mcp-server.js"]
    }
  }
}
```

Ask questions like "How do I implement authentication?" and Claude will automatically query the wiki for context.

## Development Philosophy

This project practices **wiki-driven development**:

1. Build a feature
2. Run the wiki generator on the codebase
3. Read the generated documentation
4. If unclear, improve the documentation system
5. Proceed to next feature

**The quality of auto-generated documentation validates the quality of the system itself.** This creates a virtuous cycle where better documentation tools produce better documentation, enabling more confident development.

## How It Works

1. **Repository Analysis** â€“ Fingerprints structure and patterns
2. **Agent Dispatch** â€“ Routes analysis to specialized LLM agents
3. **Content Generation** â€“ Each agent generates documentation for its domain
4. **Quality Assurance** â€“ Progressive JSON repair and response validation
5. **Assembly** â€“ Organizes content into category-based structure
6. **Linking** â€“ Discovers mentions and injects cross-page hyperlinks
7. **Index Generation** â€“ Creates navigable table of contents

This pipeline ensures comprehensive, interconnected, and maintainable documentation at scale.

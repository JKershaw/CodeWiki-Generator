# Document Mapping Configuration

## Purpose and Overview

The document mapping configuration system provides a declarative way to define how source documents are transformed and ingested into the wiki structure. It maps individual source files to their target wiki categories, document types, and file paths, enabling automated bulk processing of existing documentation into structured wiki content.

## Key Functionality

The configuration is centered around the `META_DOCS` constant, which contains an array of mapping objects that specify:

- **Source file paths** - Location of original documentation files
- **Wiki categories** - Target categorization within the wiki structure  
- **Document types** - Classification of content (guide, concept, component, etc.)
- **Target paths** - Destination file names and locations in the wiki

The system processes these mappings through the `ingestDocument` function, which:
1. Reads the source document content
2. Processes it through the MetaDocumentIngestionAgent
3. Generates appropriate frontmatter with category and type metadata
4. Writes the transformed content to the specified wiki location

The CLI interface (`main` function) enables selective processing by document name or bulk ingestion of all configured mappings, with progress reporting and error handling.

## Relationships

This component serves as the configuration layer for the [meta-documentation ingestion system](../concepts/meta-documentation-ingestion-system.md):

- **Depends on** `MetaDocumentIngestionAgent` for AI-powered document processing and frontmatter generation
- **Uses** `WikiManager` for file system operations and wiki page management
- **Integrates with** the existing wiki structure by respecting established categorization and cross-referencing patterns
- **Part of** the broader agent-based documentation automation system

## Usage Example

```javascript
// Adding a new document mapping to META_DOCS configuration
const META_DOCS = [
  {
    sourceFile: 'docs/api-reference.md',
    category: 'component',
    type: 'reference',
    targetPath: 'api-reference.md'
  }
];

// CLI usage for processing specific document
// node ingest-meta-docs.js api-reference

// Bulk processing all configured documents  
// node ingest-meta-docs.js
```

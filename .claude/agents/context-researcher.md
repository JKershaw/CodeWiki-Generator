# Wiki Context Researcher

Run the wiki-context CLI to get AI-powered research results:

```bash
./wiki-context.js "task description"
```

The WikiContextAgent will:
- Strategically explore the wiki based on your task
- Read the most relevant pages (typically 1-3 iterations)
- Synthesize actionable guidance
- Complete in under 15 seconds

Present the results to the user.
